# 税务优化计算服务 · 需求规格（v1.1）

> 按 PROJECT_IRON_RULES_SPEC Rule 9 维护：spec 先行，实现对齐 spec，交付反馈回 spec。
> 状态标记：✅ 已实现 · ⏳ 计划 · ❓ 待用户决策

## 1. 目标与角色

面向中国小规模纳税人企业主（首要用户：上海，老板本人 = 方案中的 A），提供个人收入 + 企业税负的一体化优化测算服务。

## 2. 功能需求

### 2.1 计算引擎 ✅
- 增值税：按季开票额判定免税（季度不含税销售额 ≤ 免税额度，默认 30 万），免税季价税全额确认收入；征收率、附加税率可调；按季逐季四舍五入（申报口径）。
- 社保公积金：单人级缴费基数（顶格/按工资快捷）、公司/个人社保比例、基础公积金 5–7% + 补充 0–5%（单位个人对等）。
- 个税：年度累进（税率表可编辑）、基本减除（默认 6 万/年）、六项专项附加扣除（月额可改）、年终奖单独计税（按月换算表可编辑）+ 工资/奖金拆分暴力寻优（500 元步长 + 档位边界候选）。
- 企业所得税：小微档（≤300 万 × 5%）/ 标准档（25%），附加税税前扣除。
- 股息分红 20% 个税（可选全额分红）。
- 家庭收益归集：成员级「计入家庭」标记 + 报销 + 公积金双边 + 分红净额。

### 2.2 基础数据服务 ✅
- SQLite（node:sqlite，零外部依赖）存储全国主要城市：社保/公积金基数上下限、社平工资、参考比例、生效期、官方来源、口径说明。
- 一键同步：幂等可重入 —— (code, period) 主键 upsert、逐字段比对跳过无变化、整体事务、失败回滚、sync_log 审计（版本/来源/增改数/状态）。
- 数据源：内置 data/base-data.json（版本化）；支持远程 https JSON 数据源（POST /api/sync {url}）。
- 年度更新流程：更新 base-data.json 的 version + 新 period 记录 → 一键同步；历史年度保留（?all=1）。

### 2.3 参数模块化 ✅
- 可编辑：个税税率表、年终奖税率表、企业所得税档位、增值税免税额度、基本减除、股息税率。
- 存储：全局默认（user_id=0）+ 用户级覆盖，可重置；参数随方案一并存档，实时生效。

### 2.4 体系化系统 ✅
- 多用户：创建/切换（唯一用户名；语言、默认地区随用户保存）。
- 方案（plan/profile）：完整状态快照（含员工、参数、开票节奏、快照集），按 (user_id, name) upsert，保存/另存/切换/删除。
- 会话内快照对比表（多方案并列关键指标）。
- 实时预览：任何输入变更即时全量重算。
- i18n：中英文一键切换，全界面（含动态渲染）生效。
- 离线降级：服务器不可达时前端以内置默认参数单机运行并提示。

### 2.5 企业版服务能力（边界）
- ✅ 多用户、REST API、数据审计、参数分级（全局/用户）。
- ⏳ 认证鉴权（当前无 auth，仅限本机/内网部署）、角色权限、HTTPS。
- ❓ `enforce_english_ui_only`（PROJECT_LLM_REQUIREMENTS.json）与用户明确要求的中英双语冲突 —— 当前按用户直接指令保留双语。

### 2.6 统一运维入口 ops.py ✅（v1.1 新增）
- 生命周期：init / start / stop（PID 优先、端口兜底）/ restart（默认命令）/ status / dev（--watch）/ logs。
- 部署：deploy = macOS launchd 常驻（开机自启、KeepAlive 崩溃拉起）；undeploy 撤销；其他平台提示自行托管。
- 数据与调整：sync（服务在跑走 API、停服直连 SQLite）、param get/set/reset（分用户税制参数）、config（ops.config.json：端口等）、backup（sqlite 在线备份 API）、restore（完整性校验→停服→恢复→按原状态拉起）。
- 质量：test（tests/regression.js，19 项断言）、doctor（node/端口/DB 完整性/submodule/数据集）。
- 卸载：uninstall [--purge --yes]，--purge 删数据库与备份且需确认。
- 约束：仅 Python 标准库；WAL 库只读打开需 ro → immutable → plain 回退（open_db_ro）。

## 3. 非功能需求
- 零 npm 依赖（Node ≥ 22.5 内置能力）；单命令启动 `node server.js`。
- 计算正确性以回归脚本护栏（对照用户手算基准 + 参数可调性 + 优化器最优性），页面 headless 冒烟。
- 免责与合规提示常驻：政策时效/地区差异、亲属实习真实用工、房租报销与租金扣除互斥。
